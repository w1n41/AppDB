Инструкция для подключения базы данных к приложению.
-----------------------------------------------------------------------------------------------------------------------
ВНИМАНИЕ! Важно сделать всё точно по инструкции, иначе приложение будет работать не корректно, либо работать не будет вовсе!

Шаг 1:
	В первую очередь нужно создать базу данных, выполнив запросы из архива в таком порядке: 1. CreateDB.sql, 2. Tables.sql, 3.Filling Tables.sql.
Шаг 2: 
	После создания базы данных её таблиц и их заполнения, открываем приложение через VisualStudio. Уже в VS после открытия приложения открываем вкладку "Средства", и там выбираем пункт "Подключиться к базе данных". Далее выполняем всё по скриншотам по возрастанию.
Шаг 3: 
	После выбора источника данных, появляется окно подключения к серверу (Скриншот 2), в нём же есть текстовое поле "Имя сервера". Туда в свою очередь мы копируем имя сервера из SSMS (Пример показан на скриншоте 4).
Шаг 4:
	После вставки имени сервера в соответствующее поле, опускаемся ниже и выбираем свою базу данных в выпадающем списке поля "Выберите или введите имя базы данных" (Скриншот 5, 6). 
-----------------------------------------------------------------------------------------------------------------------
После данных манипуляций, программа должна исправно начать свою работу, не выдавая никаких сбоев. Приятного пользования! <3
